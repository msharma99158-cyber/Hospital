# Hospital Management System

## 📌 Project Overview

The Hospital Management System is a web-based application designed to help hospitals and patients manage healthcare services efficiently, especially during emergency situations.

This system is specially developed for critical medical cases where immediate treatment is required but the patient's family members are not physically available in the hospital to provide consent. The platform allows hospitals to send an online emergency agreement form to family members so treatment can begin without unnecessary delays.

The project also includes features like ambulance booking, bed booking, appointment scheduling, and booking management to improve hospital operations and patient care.

---

## 🚀 Features

- 🚑 Emergency online consent/agreement form
- 🚑 Ambulance booking system
- 🛏️ Hospital bed booking
- 📅 Appointment booking
- 📋 My Bookings section for patients
- 👨‍💼 Admin panel for hospital management
- 👨‍⚕️ Patient management
- 🔐 Secure login and authentication
- 📂 Hospital record management
- ⚡ Quick response during emergency situations

## 🎯 Objective

The main objective of this project is to reduce delays in emergency treatment and simplify hospital operations through digital healthcare services.

The system helps hospitals communicate quickly with patients' family members and provides important hospital services online in a secure and efficient manner.
